Unir Líneas

Dado un texto de entrada en múltiples líneas puede unirlas con un separador. Puede determinar el separador de los elementos a unir.

También puede asignar una expresión regular para filtrar las líneas que se unen.

Si lo desea puede asignar una expresión regular (sin /\) para filtrar las líneas que se unen.
